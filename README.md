# nuxt3-template
 Nuxt3+vue3 + NaiveUI + UnoCSS + Pinia + VueUse + useFetch pc+移动端 模板
## 步骤

安装依赖：

```bash
# pnpm
pnpm install
```

## 开发环境

启动开发环境 `http://localhost:9527`:

```bash
# pnpm
pnpm run dev
```

## 生产环境

构建生产环境的应用程序：

```bash
# pnpm
pnpm run build
```

本地预览生产版本：

```bash
# pnpm
pnpm run preview
```
**注意：**
1. 所有文件命名方式遵循小驼峰命名
2. 组件不再写在pages中，在components中创建同pages文件名的文件夹书写组件
